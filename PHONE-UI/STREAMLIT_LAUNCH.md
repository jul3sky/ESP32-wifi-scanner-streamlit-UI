# HOW TO LAUNCH YOUR STREAMLIT APP (WINDOWS 11)
============================================================================================

- Create the project folder on your computer.
- Put your streamlit code into the folder.
- Make sure that your code points to correct serial port
  `ser = serial.Serial("COM3", 115200`.
- Make sure all the required libraries are installed.
- See if your Streamlit is installed too.
- Run CMD inside the project folder.
- Command: `streamlit run <filename.py>`
  Steamlit will run the app locally.
